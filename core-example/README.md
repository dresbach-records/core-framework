# Exemplo Completo CORE: Site Institucional + Dashboard

Este projeto é uma demonstração completa e funcional dos princípios do framework CORE.

### O que este exemplo prova:

- ✔️ **Site institucional real:** Páginas públicas como home e serviços.
- ✔️ **Dashboard funcional:** Uma área privada acessível após um "login".
- ✔️ **Server-first:** Toda a lógica e estado residem no servidor.
- ✔️ **HTML como contrato:** O servidor responde com HTML, não com JSON para a UI.
- ✔️ **Estado no servidor:** O estado da aplicação é gerenciado inteiramente no backend.
- ✔️ **Sem SPA:** A aplicação segue o modelo tradicional de requisição-resposta da web.
- ✔️ **Fácil de entender:** A arquitetura é clara e o fluxo de dados é explícito.
- ✔️ **Fácil de evoluir:** A base é sólida para adicionar funcionalidades complexas como autenticação real, banco de dados e WebSockets, sem precisar reescrever tudo.

Este exemplo fecha o CORE como um framework viável e produtivo, não apenas conceitual.
